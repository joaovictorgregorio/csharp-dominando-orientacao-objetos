## 📝 About This Repository

This repository contains the practical code developed during the course. Its main goal is to strengthen Object-Oriented Programming skills in C#, with focus on inheritance, interfaces, encapsulation, polymorphism, and method overriding.

## 💻 How to Run the Code

Since this project was built using the **.NET CLI**, you can easily run it from your terminal.

1. Ensure you have the [.NET SDK](https://dotnet.microsoft.com/download) installed.
2. Clone this repository and navigate to the desired project folder.
3. Run the following command in the terminal:

```bash
dotnet run
```

## 📂 Repository Structure

| Path                           | Description                                               |
| ------------------------------ | --------------------------------------------------------- |
| ScreenSound/Program.cs         | Main entry point for the ScreenSound project.             |
| ScreenSound/Models/Band.cs     | `Band` class with notes, albums, and average calculation. |
| ScreenSound/Models/Album.cs    | `Album` class containing songs and total duration logic.  |
| ScreenSound/Models/Music.cs    | `Music` class with technical and descriptive metadata.    |
| ScreenSound/ScreenSound.csproj | Project file for ScreenSound.                             |
| ScreenSound/ScreenSound.sln    | Solution file for opening and managing the project in IDE |
| ScreenSound/bin/               | Compiled output generated at build/run time.              |
| ScreenSound/obj/               | Intermediate build artifacts generated by .NET SDK.       |
